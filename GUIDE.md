# Team Share 사용 가이드

> 팀원들과 자료를 공유하고 실시간으로 소통하는 협업 플랫폼

---

## 1. 시작하기 (처음 사용자)

### 접속
- **주소**: https://team-share-three.vercel.app
- 브라우저(크롬, 사파리 등)에서 위 주소로 접속

### 회원가입
1. **회원가입** 버튼 클릭
2. 이름, 이메일, 비밀번호 입력
3. 완료 → 자동으로 로그인됩니다

### 팀 합류
- 관리자에게 받은 **초대 코드**를 입력하면 팀에 합류됩니다
- 여러 팀에 동시에 소속될 수 있습니다

---

## 2. 앱처럼 설치하기 (홈 화면 추가)

설치하면 앱처럼 전체 화면으로 사용할 수 있습니다. (용량 거의 안 차지합니다)

### 아이폰 (Safari)
1. Safari로 https://team-share-three.vercel.app 접속
2. 하단 **공유 버튼** (□↑) 터치
3. **"홈 화면에 추가"** 선택
4. **추가** 터치
5. 홈 화면에 **Team Share** 아이콘이 생깁니다

### 안드로이드 (Chrome)
1. Chrome으로 https://team-share-three.vercel.app 접속
2. 주소창 오른쪽 **⋮ 메뉴** 터치
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
4. **설치** 터치
5. 홈 화면에 **Team Share** 아이콘이 생깁니다

### PC (Chrome)
1. Chrome으로 접속
2. 주소창 오른쪽에 **설치(⊕)** 아이콘 클릭
3. **설치** 클릭

---

## 3. 주요 기능

### 📊 대시보드
- 로그인하면 가장 먼저 보이는 화면
- 팀 활동 요약, 다가오는 기한, 최근 파일 등을 한눈에 확인

### 💬 채팅
- 팀별 채팅방에서 실시간 대화
- 텍스트, 이미지, 파일 전송 가능
- 메시지를 **길게 누르면** → 할일/의사결정/아이디어로 변환 가능

### 📁 자료 공유
- 파일/이미지 업로드 (최대 10MB)
- 드래그 & 드롭으로 간편 업로드
- 태그를 추가해서 분류, 검색으로 빠르게 찾기

### 📌 액션
- 채팅에서 변환된 할일/의사결정/아이디어 관리
- 담당자 지정, 기한 설정, 상태 변경

### 🔔 알림
- 채팅, 파일 업로드, 액션 수정 시 자동 알림
- 설정에서 알림 유형별 on/off 가능

### 👥 팀 관리
- 팀 생성 (초대 코드 자동 생성)
- 초대 코드로 팀원 초대
- 여러 팀 전환 가능

---

## 4. 화면 안내

### 모바일
- **상단**: 팀 이름 (터치하면 팀 전환)
- **하단 메뉴바**: 대시보드 · 채팅 · 자료 · 액션 · 알림 · 팀 · 설정

### PC
- **왼쪽 사이드바**: 팀 전환 + 메뉴 + 프로필
- **오른쪽**: 메인 콘텐츠

---

## 5. 자주 묻는 질문

**Q: 비밀번호를 잊었어요**
→ 현재 비밀번호 재설정 기능은 없습니다. 관리자에게 문의해주세요.

**Q: 올린 파일은 어디에 저장되나요?**
→ 클라우드(Cloudinary)에 안전하게 저장됩니다. 앱에서 언제든 다운로드 가능합니다.

**Q: 팀을 나가고 싶어요**
→ 설정 또는 팀 관리에서 "팀 나가기"를 선택하세요.

**Q: 알림이 너무 많아요**
→ 설정에서 알림 유형별로 끌 수 있습니다.

**Q: 다른 팀에도 가입할 수 있나요?**
→ 네, 여러 팀에 동시에 소속될 수 있고 상단/사이드바에서 팀을 전환합니다.

---

## 6. 관리자용

### 팀 만들기
1. 팀 관리 → **팀 생성**
2. 팀 이름 입력 → 초대 코드가 자동 생성됩니다
3. 초대 코드를 팀원들에게 공유하세요

### 팀 관리
- 멤버 추방, 초대 코드 재생성, 팀 이름 변경 가능
- 팀 생성자가 관리자입니다
